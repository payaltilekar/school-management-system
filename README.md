# 🎓 School Management System

A responsive **School Management System** built using **Python**, **Object-Oriented Programming (OOP)**, **JSON**, **File Handling**, and **Streamlit**.

This project includes both a **console-based application** and a **Streamlit web application** for managing students and teachers.

---

## ✨ Features

### 👨‍🎓 Student Management
- Register new students
- Add subject-wise grades
- View student details
- Calculate average marks
- Email validation

### 👩‍🏫 Teacher Management
- Register teachers
- View teacher details
- Store subject information

### 💾 Data Storage
- Store student and teacher records in a JSON file
- Persistent data even after closing the application

### 🌐 Streamlit Web Interface
- Responsive dashboard
- Student leaderboard
- Teacher management
- Modern user interface
- School statistics

---

## 🚀 How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/payaltilekar/school-management-system.git
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the Console Version

```bash
python main.py
```

### 4. Run the Streamlit Version

```bash
streamlit run app.py
```
